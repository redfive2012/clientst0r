# What is ClientSt0r?

ClientSt0r is an **open-source documentation platform built for Managed Service Providers (MSPs)** and IT teams.

It centralizes operational knowledge and client information into a structured, searchable system designed to support daily service delivery.

## Typical Use Cases

- Client documentation
- Asset tracking and rack management
- Credential storage (AES-GCM encrypted password vault)
- SOPs and runbooks
- Vendor and warranty tracking
- Internal knowledge base
- Fleet and vehicle management
- Network infrastructure (IPAM, VLANs, subnets)

## Who It's For

- Managed Service Providers
- Internal IT departments
- Homelab operators needing structured documentation

## Technology

Built with Django (Python), supports MariaDB, runs on Linux. One-line installer, web-based auto-update.
