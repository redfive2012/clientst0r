# MSP Documentation with ClientSt0r

Effective MSP operations rely on accurate, accessible documentation. ClientSt0r provides a structured platform for organizing all client and infrastructure knowledge.

## What ClientSt0r Helps Document

### Clients & Organizations
- Multi-organization support with complete data isolation
- Per-client documentation spaces
- Contact management

### Assets & Infrastructure
- Device and asset tracking with full lifecycle management
- Interactive rack diagrams with drag-and-drop
- Wall-mounted board layouts for network closets
- IPAM (IP address management) with subnet tracking
- VLAN documentation
- Patch panel management with visual cable paths

### Security
- AES-GCM encrypted password vault
- Personal vaults per user
- Password breach detection (HaveIBeenPwned integration)
- Encrypted backups

### Knowledge Base
- Per-organization documentation with version control
- Global MSP knowledge base (staff-only)
- Draw.io diagram integration
- Floor plan import (MagicPlan)
- SOPs and runbooks

### Monitoring & Expiration Tracking
- Website and SSL certificate uptime monitoring
- Domain and SSL expiration alerts
- Credential expiration tracking

### Fleet Management
- Service vehicle tracking
- Mileage, maintenance, and fuel logs
- Interactive damage reports with vehicle diagrams
- Vehicle inventory management

### Integrations
- 8 PSA providers (ConnectWise, Autotask, HaloPSA, Kaseya BMS, Syncro, Freshservice, Zendesk, ITFlow)
- 5 RMM providers (Tactical RMM, NinjaOne, Datto, Atera, CW Automate)
- REST API v1 and GraphQL API v2

## Goal

Reduce tribal knowledge risk, improve onboarding speed, and enable consistent service delivery across the team.
