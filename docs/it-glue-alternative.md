# Client St0r as an IT Glue Alternative

Many MSPs compare Client St0r when they want a self-hosted solution or when they're specifically looking for an open-source alternative to IT Glue.

## High-level comparison

| Category      | Client St0r                           | IT Glue                         |
| ------------- | ------------------------------------- | ------------------------------- |
| License       | Open-source                           | Proprietary                     |
| Hosting       | Self-hosted                           | SaaS                            |
| Data control  | You control infrastructure & database | Vendor-hosted                   |
| Customization | Modify/extend via code                | Limited to vendor features      |
| Support model | Community + self-managed              | Vendor support (plan-dependent) |

## Notes

* Client St0r is intended for MSP documentation workflows. Exact feature parity depends on your deployment and configuration.
* If you rely on specific IT Glue integrations or workflows, validate those requirements before migrating.
